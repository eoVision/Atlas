CESIUM

  • 3D Marker auf GlobusDieses Beispiel zeigt verschiedene Möglichkeiten auf, wie Orte auf dem Globus markiert werden können. Es werden Punkte, Polygone und 3D-Modelle verwendet, welche jeweils auch mit Text-Labels versehen werden können. Bei Bewegen der Maus verändert sich der Mauszeiger und weist so darauf hin, dass die Objekte angeklickt werden können. Ein Klick auf ein Objekt auf dem Globus führt direkt auf eine verlinkte Webseite. Sowohl die Position der Objekte auf dem Globus als auch die hinterlegten Links sind im JavaScript-Code manuell erzeugt worden.
  • Suchleiste

    Anstelle von im Code definierten Geometrien wie in Beispiel 1, werden in diesem Beispiel GeoJSON-Dateien verwendet verwendet (aus QGIS exportiert) um Orte auf dem Globus anzuzeigen. Das Clustering von Polygonen wird von Cesium nicht unterstützt (lediglich Punkte, Billboards und Labels). Um dennoch zur besseren Orientierung eine Cluster-Funktion zu haben, wurde für jedes Polygon der Mittelpunkt (centroid) berechnet und diese Punkte und deren Labels werden geclustert. So kann der Benutzer durch die Nummernanzeige der Cluster und durch die Marker der Punkte übersichtlich alle Polygone auf dem Globus finden. Sowohl die Polygone als auch die Marker und Label der Punkte führen beim Anklicken zu der hinterlegten URL. Der Link zur Webseite für jedes Polygon kann als Attribut in der GeoJSOn-Datei hinterlegt werden und beim Klick abgefragt werden. Zudem sind in diesem Beispiel Zoomstufen eingebaut. So werden sowohl die Punkte als auch die Polygone erst ab einer bestimmten Kamerahöhe eingeblendet.
